## Steps to reproduce:
* Configure [python](https://www.python.org/) in your system
* Configure [pip](https://pip.pypa.io/en/stable/) as well in your system
* [Download](https://github.community/t/how-can-i-download-a-specific-folder-from-a-github-repo/278) this repository into your system as a folder
* Now open terminal in that folder and type `pip install -r requirements.txt` in the shell
* Now type `python bot.py` into the shell and wait for a message saying "Logged in as Anzen#3274" and you are good to go!
